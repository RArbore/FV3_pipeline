# Input netCDF
## Dimensions
- plev: Number of pressure levels
- time: Number of times
- sample: Number of samples
- SZA: Number of SZAs
- VZA: Number of VZAs
- RAA: Number of RAAs

## Variables
- lat(sample): Latitude (degrees north)
- lon(sample): Longitude (degrees east)
- plev(plev): Pressure Level (millibar, lowest pressure level is first, last entry is surface)
- time(time): Hours since August 1st, 2016 (UTC)
- q_plev(plev, sample): Specific humidity (kg water vapor / kg air, per pressure level)
- t_plev(plev, sample): Temperature (kelvin, per pressure level)
- qi_plev(plev, sample): In-cloud ice water concentration (kg ice / kg air, per pressure level)
- ql_plev(plev, sample): In-cloud liquid water concentration (kg liquid / kg air, per pressure level)
- ts(sample): Surface skin temperature (kelvin)
- ps(sample): Surface pressure (pascals)
- cldc(sample): Total column cloud cover (%)
- fsus(sample): Surface upwelling shortwave broadband radiative flux (watt / meter ^ 2)
- fsds(sample): Surface downwelling shortwave broadband radiative flux (watt / meter ^ 2)
- fsut(sample): Top of atmosphere upwelling shortwave broadband radiative flux (watt / meter ^ 2)
- fsdt(sample): Top of atmosphere downwelling shortwave broadband radiative flux (watt / meter ^ 2)
- SZA(SZA): Solar Zenith Angle (degrees, 90 is horizon)
- VZA(VZA): Viewing Zenith Angle (degrees, 90 is horizon)
- RAA(RAA): Relative Azimuth Angle (degrees)

# Output netCDF
## Dimensions
- num_cases: Number of unique profiles passed in
- num_szas: Number of solar zenith angles to run MODTRAN over
  - NOTE: there is one MODTRAN case per num_cases and per num_szas; almost every variable's first two dimensions will be num_cases and num_szas
- profiles_len: Length of profiles passed into MODTRAN
- num_wavenumbers: Number of wavenumbers in MODTRAN output
- num_input_geometries: Number of input geometries used (combinations of obszen and azinp)
- num_path_geometries: Number of geometries in output column absorber amounts
- num_toa_radiances: Number of toa_radiances per wavelength
- num_wavelengths: Number of wavelengths varied over in toa_radiances
- fluxes_per_index: Number of fluxes per wavelength (number of altitudes considered)
  - NOTE: It is expected behavior that num_input_geometries = num_toa_radiances and num_path_geometries = fluxes_per_index
- los_paths/num_los_paths: Number of line-of-sight paths
## Variables
- szas(num_szas): Solar Zenith Angle (degrees, 90 is horizon)
- prof_pressure(num_cases, num_szas, profiles_len): Pressure (millibar, highest pressure is first, lowest pressure is last, must monotically decrease)
- prof_temperature(num_cases, num_szas, profiles_len): Temperature (kelvin)
- prof_h2o(num_cases, num_szas, profiles_len): H2O mixing ratio (ppmv)
- prof_co2(num_cases, num_szas, profiles_len): CO2 mixing ratio (ppmv)
- prof_o3(num_cases, num_szas, profiles_len): O3 mixing ratio (ppmv)
- prof_n2o(num_cases, num_szas, profiles_len): N2O mixing ratio (ppmv)
- prof_co(num_cases, num_szas, profiles_len): CO mixing ratio (ppmv)
- prof_ch4(num_cases, num_szas, profiles_len): CH4 mixing ratio (ppmv)
- toa_rad(num_cases, num_szas, num_wavenumbers): Top of atmosphere band model radiance (watt / cm^2 / sr, per wavenumber)\*
- ref_sol(num_cases, num_szas, num_wavenumbers): Channel in-band irradiance, computed by band model (watt / cm^2, per wavenumber)\*
- sol_at_obs(num_cases, num_szas, num_wavenumbers): Channel in-band top of atmosphere solar irradiance, transmitted to observer altitude (watt / cm^2, per wavenumber)\*
- depth(num_cases, num_szas, num_wavenumbers): Optical depth (unitless, per wavenumber)
- obszen(num_cases, num_szas, num_input_geometries): Input observer zenith angle (degrees, 90 is horizon)
- azinp(num_cases, num_szas, num_input_geometries): Input azimuth angle (degrees)
- day(num_cases, num_szas): Day of year (julian day)
- temp(num_cases, num_szas): Area-averaged ground temperature (kelvin)
- toa_radiances_wavelengths(num_cases, num_szas, num_toa_radiances, num_wavelengths): Wavelengths corresponds to toa_radiances (nm)
- toa_radiances(num_cases, num_szas, num_toa_radiances, num_wavelengths): Output top of atmosphere spectral radiances, convolved with 2nm full-width half-maximum triangular slit function (uwatt / cm^2 / sr / nm)
- upward_diffuse(num_cases, num_szas, num_toa_radiances): Upward diffuse flux (watts / cm^2 / nm, first entry corresponds to the lowest wavelength and lowest altitude level)
- downward_diffuse(num_cases, num_szas, num_toa_radiances): Downward diffuse flux (watts / cm^2 / nm, first entry corresponds to the lowest wavelength and lowest altitude level)
- direct_solar(num_cases, num_szas, num_wavelengths, fluxes_per_index): Direct solar flux (watts / cm^2 / nm, first entry corresponds to the lowest wavelength and lowest altitude level)
- toa_radiances_vis(num_cases, num_szas, fluxes_per_index): Spectrally integrated over VIS wavelength range output top of atmosphere spectral radiances, convolved with 2nm full-width half-maximum triangular slit function (uwatt / cm^2 / sr)
- upward_diffuse_vis(num_cases, num_szas, fluxes_per_index): Spectrally integrated over VIS wavelength range upward diffuse flux (watts / cm^2, first entry corresponds to the lowest altitude level)
- downward_diffuse_vis(num_cases, num_szas, num_wavelengths, fluxes_per_index): Spectrally integrated over VIS wavelength range downward diffuse flux (watts / cm^2, first entry corresponds to the lowest altitude level)
- direct_solar_vis(num_cases, num_szas, fluxes_per_index): Spectrally integrated over VIS wavelength range direct solar flux (watts / cm^2, first entry corresponds to the lowest altitude level)
- toa_radiances_nir(num_cases, num_szas, fluxes_per_index): Spectrally integrated over NIR wavelength range output top of atmosphere spectral radiances, convolved with 2nm full-width half-maximum triangular slit function (uwatt / cm^2 / sr)
- upward_diffuse_nir(num_cases, num_szas, num_wavelengths, fluxes_per_index): Spectrally integrated over NIR wavelength range upward diffuse flux (watts / cm^2, first entry corresponds to the lowest altitude level)
- downward_diffuse_nir(num_cases, num_szas, fluxes_per_index): Spectrally integrated over NIR wavelength range downward diffuse flux (watts / cm^2, first entry corresponds to the lowest altitude level)
- direct_solar_nir(num_cases, num_szas, fluxes_per_index): Spectrally integrated over NIR wavelength range direct solar flux (watts / cm^2, first entry corresponds to the lowest altitude level)
